RedMine

RedMine is a modern issue tracking system set up by AAFC's Metagenomics, Microbiology and Biodiversity (MBB) department. It supports the majority of AAFC Bioinformatics projects and has been extended to support CFIA needs. It will be the primary point of contact relating to Bioinformatics support for OLC SPAdes Assembly Pipeline and the CFIA Genomics Web Portal. This tool compliments the SharePoint ColLABarotory designed for wet lab procedures.

Accessing RedMine

Navigate to http://redmine.biodiversity.agr.gc.ca Click Sign In (Top Right Corner)
Login with your AAFC or CFIA credentials
First time users will need to ensure their contact email is correct

CFIA Genomics represents our corner of RedMine. The two important sub projects are:

OLC Assembly Pipeline
This project lead by Dr. Adam Koziol will assemble and analyze bacterial genomes generated by Illumina MiSeqs.

CFIA Genomics Portal
This project lead by Jackson Eyres provides a simplified interface for bioinformatic web apps and will support the OLC Assembly Pipeline.

Clicking a project’s name displays its overview:
Brief description
Support staff responsibilities
Existing issues (bugs, features, and support requests)

Modifying an issue will send an email notification to its watchers To reduce email notifications, it is recommended you change settings under My Account, to “Only for things I watch or I'm involved in”

General Usage

File Uploading

Projects

Projects are the primary form of interacting with the various tools of the website. They represent a collection of genomes, either raw reads (FastQ) or assemblies (Fasta). Projects are seperated into two categories, "Custom FastQ" and "Custom Fasta".

Creating A New Project

After succesfully uploading some genomes using File Upload, the files are available to be added to a project. It is recommended that every genome in a project be the same organism.

FastQ Projects use paired end files, meaning that every sample must be composed of two files (forward and reverse reads). The server expects files to be in the naming scheme of Sample_R1_001.fastq.gz and Sample_R2_001.fastq.gz.
